# Библиотека Теней

Магическая библиотека — сайт-витрина для твоих книг и текстов.

## Как добавить книгу

Создай файл в папке `src/content/books/` с расширением `.md`:

```markdown
---
title: Название книги
shelf: 1
position: 3
color: "#8B1A1A"
spineColor: "#5a0f0f"
textColor: "#ffd700"
---

Текст книги здесь. Можно использовать **жирный**, *курсив*, заголовки # и т.д.

Новый абзац — просто пустая строка между текстом.
```

### Параметры:

| Поле | Описание |
|------|----------|
| `title` | Название (отображается на корешке и в заголовке) |
| `shelf` | Номер полки от 1 до 48 (12 полок × 4 стеллажа) |
| `position` | Место на полке от 1 до 11 |
| `color` | Цвет обложки (hex) |
| `spineColor` | Цвет боковины корешка — обычно темнее `color` |
| `textColor` | Цвет надписи на корешке |

### Расположение стеллажей:
- Стеллаж 1: полки 1–12
- Стеллаж 2: полки 13–24
- Стеллаж 3: полки 25–36
- Стеллаж 4: полки 37–48

## Локальный запуск

```bash
npm install
npm run dev
```

## Деплой на Vercel

1. Загрузи проект на GitHub
2. Зайди на [vercel.com](https://vercel.com)
3. Нажми "Add New Project" → выбери репозиторий
4. Vercel автоматически определит SvelteKit — жми Deploy
5. После каждого добавления книги — просто делай `git push`

## Управление читалкой

- **Клик по правой половине** — следующая страница
- **Клик по левой половине** — предыдущая страница
- **→ / Пробел** — следующая страница
- **←** — предыдущая страница
- **Escape** — закрыть книгу
- **Точки внизу** — быстрый переход к странице
