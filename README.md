## Добавление книги

Создать файл в папке `src/content/books/` с расширением `.md`:

```markdown
---
title: Название книги
shelf: 1
position: 3
color: "#8B1A1A"
spineColor: "#5a0f0f"
textColor: "#ffd700"
---

Текст книги здесь. 

**жирный текст**
*курсив*
***жирный курсив***

# Большой заголовок
## Средний заголовок
### Маленький заголовок

Разделитель (отображается как ✦ ✦ ✦): ---

> Это цитата

~~зачёркнутый~~

Пробел между абзацами: ⁠~~~
```

### Параметры:

| Поле | Описание |
|------|----------|
| `title` | Название (отображается в заголовке) |
| `shelf` | Номер полки от 1 до 48 (12 полок × 4 стеллажа) |
| `position` | Место на полке от 1 до 11 |
| `color` | Цвет обложки (hex) |
| `spineColor` | Цвет боковины корешка |
| `textColor` | Цвет надписи на корешке |

### Расположение стеллажей:
- Стеллаж 1: полки 1–12
- Стеллаж 2: полки 13–24
- Стеллаж 3: полки 25–36
- Стеллаж 4: полки 37–48
