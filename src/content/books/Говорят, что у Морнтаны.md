Говорят, что у Морнтаны
И евойных аварей
Есть по сотне грам сметаны
И по паре тыщ рублей
<br>
Есть у них своя печатка
Есть кумарошная грядка
Есть штандарты и гербы
Есть трава и есть грибы
Есть дворцы и есть беседки
Есть деды и малолетки
Есть эльфийский паритет
Даже есть нейтралитет
Есть у а́варей деньжата
Живут дорого богато
Есть руничное письмо
На кой хер нужно оно?
Есть наемный фаразтил
Есть мифриловый настил
Жены есть и есть мужья
Даже порох для ружья
Есть история в веках
Есть и башни в облаках
Есть закуска и вино
Есть мышиное говно
Есть у них такая штука
Как прицел с резьбой для лука
Есть талант и есть прогресс
Есть герой и мракобес
Есть столовые приборы
Есть в подвалах Берифоры
<br>
Есть у а́варей секрет -
Аваре́й давно уж нет
